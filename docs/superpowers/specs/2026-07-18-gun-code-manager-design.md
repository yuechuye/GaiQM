# 三角洲改枪码管理网站 — 设计文档

## 概述

一个托管在 GitHub Pages 上的纯静态网站，用于展示、筛选和复制三角洲(Delta Force)游戏的改枪码。数据存储在 JSON 文件中，通过 git 进行版本管理。

## 技术选型

- 纯静态 HTML + CSS + JavaScript，无框架依赖
- 数据存储：`data.json` 文件
- 托管：GitHub Pages

## 数据结构

```json
{
  "guns": [
    {
      "name": "M4A1",
      "codes": [
        {
          "code": "xxx-xxx-xxx",
          "description": "高精度远程配置",
          "value": 15000
        }
      ]
    }
  ]
}
```

- `guns[].name` — 枪械名称
- `guns[].codes[].code` — 改枪码
- `guns[].codes[].description` — 描述
- `guns[].codes[].value` — 价值（数值）

一种枪械可以有多个改枪码（`codes` 数组）。

## 页面布局

单页面应用，从上到下分为：

1. **顶栏** — 网站标题 + GitHub 链接
2. **筛选栏** — 搜索框 + 枪械下拉筛选 + 价值排序按钮
3. **码列表** — 按枪械分组展示，每组显示枪名和码数量，每个码以卡片形式展示

## 功能清单

| 功能 | 描述 |
|------|------|
| 展示 | 按枪械分组展示所有改枪码 |
| 搜索 | 顶部搜索框支持模糊搜索（匹配码、描述、枪名） |
| 筛选 | 按枪械名称下拉筛选，按价值升序/降序排列 |
| 复制 | 每个码卡片有复制按钮，一键复制到剪贴板，复制后反馈"已复制" |
| 响应式 | 适配手机和桌面端 |

## 文件结构

```
/
├── index.html      # 主页面结构
├── style.css       # 样式（简洁实用风格）
├── script.js       # 交互逻辑（加载数据、筛选、复制等）
└── data.json       # 改枪码数据（手动维护，git 版本管理）
```

## 样式风格

简洁实用风格：干净白底、卡片式布局、合理间距、移动端友好。不依赖任何 CSS 框架。

## 非功能需求

- 首次加载即展示所有数据，无需额外请求
- 复制功能兼容主流浏览器（使用 Clipboard API）
- 纯静态，无任何外部依赖
